# Data Analysis

This repository contains simple tools and scripts for basic data analysis tasks.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/rkray965497/data-analysis.git
   cd data-analysis
   ```

2. Install the requirements:
   ```bash
   pip install -r requirements.txt
   ```

3. Run scripts or open the notebooks in the `notebooks/` folder.

## Contents

- `notebooks/` — Example Jupyter notebooks
- `src/` — Python scripts for data analysis
- `data/` — Example datasets

## Requirements

- Python 3.7 or higher
